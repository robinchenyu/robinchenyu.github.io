On deploying post with github pages and cryogen

Install Cryogen

Install Cryogen

lein new cryogen my-blog
cd my-blog
lein ring server

修改Cryogen配置参数

编辑resources/templates/config.edn,修改配置参数blog-prefix为空。 修改后，需要运行lein ring server，这时会自动更新resources/public目录下的文件

register and git a github pages account robinchenyu.github.io


克隆robinchenyu.github.io项目到本地


克隆项目到本地后，拷贝templates/public下的所有文件到项目中，然后测试 主页
